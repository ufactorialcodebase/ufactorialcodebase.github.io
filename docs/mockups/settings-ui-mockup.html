Changes in v2
• Top is now a collapsed Profile settings row with Edit → opens the Profile settings page (display name, email, change password, user ID). The big avatar + name are gone.
• Subscription: Current plan = "Closed beta"; usage row removed; two rows only (Current plan, Manage subscription). Upgrade to Premium moved into Manage subscription and greyed out, no price until launch — logged in docs/project/launch_checklist.md.
• Data & Privacy: new Privacy settings page (Data sharing: Usage analytics, Crash reports, AI improvement — Community insights excluded). Export / Delete remain "Coming soon".
• About: Version = 2.0.
• Preferences = Dark mode (relocated from the mobile top bar, wired to existing useTheme) + Language (English; structure ready for more locales).
• Privacy toggles aren't persisted yet — they need a small backend store before they do anything (tracked in the launch checklist).
